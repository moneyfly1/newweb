## 🔧 技术实现

### 统一组件
- **CommonDrawer** - 统一的抽屉组件
  - 响应式宽度（移动端 100%，桌面端可配置）
  - 统一的 footer 按钮
  - 加载状态支持
  - 确认/取消事件

### 转换模式
```vue
<!-- 修改前 -->
<n-modal v-model:show="showModal" preset="card" title="标题">
  <n-form>...</n-form>
  <template #footer>
    <n-button @click="showModal = false">取消</n-button>
    <n-button type="primary" @click="handleSubmit">确定</n-button>
  </template>
</n-modal>

<!-- 修改后 -->
<common-drawer
  v-model:show="showDrawer"
  title="标题"
  :width="600"
  show-footer
  :loading="saving"
  @confirm="handleSubmit"
  @cancel="showDrawer = false"
>
  <n-form>...</n-form>
</common-drawer>
```

### 代码修改
- 变量名：`showModal` → `showDrawer`
- 添加 `saving` 状态
- 导入 `CommonDrawer` 组件
- 调整 `finally` 块位置
